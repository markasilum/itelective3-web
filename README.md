### Features
#### Faculty
* Login
* Register
* Add student's attendance
* Modify student's attendance
* Delete student's attendance
* View student's attendance
* View student's attendance by date
* View class Attendance
#### Students
* Login
* Register
* View attendance
* View attendance by date
* View class attendance
* Change password
#### Administrator
* Add student's attendance
* Modify student's attendance
* Delete student's attendance
* Add users
